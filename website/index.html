David Zilli, Alan Duanmu, Garrett Hamers, Asif Aziz

About Vector

Vector is a puzzle platformer game where you must figure out the correct sequence in order to make it to the next level. The objective of the game is to solve the puzzles and to have fun doing it. For this reason, we had to find the right balance of difficulty and easiness. If the puzzles are too easy they are not entertaining, but if the puzzles are too difficult, they become frustrating. We strive for that just-challenging-enough medium for Vector.

Design Principles

  • For the beginning levels, we have added text that will give hints on how to approach the level.
  • For movement, we use both the ‘W’’A’’S’’D’ keys and the arrow keys, so that people with lots of experience with computer games and an nonnative users can both intuitively use the game.
  • To help the user understand that they can move their spite without having to learn through trial and error, we have added text to the top left part of the screen that is a short description of the movement keys.

Features

  • The game is built on HTML5 so it is playable through any modern browser.
  • It also makes it easily playable on a mobile device.
  • Gravity changing is done by mouse or touch input for usability.
